Tupelo Web Store

This is a web-accessible Tupelo Store, a repository for managed whole disk content.

This site is not intended for human consumption, but rather tool-based access. Nevertheless, some features are provided below for testing purposes.

  • Server Version
  • Server's Store UUID
  • Server's Free Disk Space
  • New Store Session Identifier
  • List Managed Disks
  • Store Tools

Logging Levels

You can update the server's own logging levels, on the fly. The server likely logs to an AMQP-accessible 'Log Monitor', so you can adjust traffic to that monitor here.
